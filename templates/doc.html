GET

Retourne en format json, les données d'un article dont l'identifiant est spécifié par le paramétre identifiant.

Codes de status http possibles

  • 200
    • L'article reféré par identifiant a été trouvé

  • 404
    • L'article spécifié par identifiant n'a pas été trouvé

GET


Retourne en format json, des données de tous les articles déjà publiés. C'est à dire les articles dont la date de publication est inférieur à la date courante.

Codes de status http possibles

  • 200
    • Succès. L'opération s'est déroulée correctement

POST

Crée un article, à partir de données reçues en format json

Codes de status http possibles

  • 200
    • L'article a été créé

  • 400
    • Mauvaise requete dû soit à un champ manquant soit à un identifiant déjà existant
